# Chrome History Cleaner Extension

A simple Chrome extension to clear browsing history for specific websites.

## Features
- Clear history for any website by domain
- Simple and clean interface
- Shows count of deleted entries

## Installation
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension folder

## Usage
1. Click the extension icon
2. Enter a domain (e.g., youtube.com)
3. Click "Clear History"

## Files
- `manifest.json` - Extension configuration
- `popup.html` - User interface
- `popup.js` - Core functionality
